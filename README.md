# WORKBOX 

### Module 
`yarn add -D workbox-precaching`
`yarn add -D esbuild`
`yarn add -D workbox-cli`
`yarn add -D workbox-routing workbox-strategies`
`yarn add -D workbox-cacheable-response`
`yarn add -D workbox-expiration`

# WORKBOX SETUP
`npx workbox-cli wizard --injectManifest`

# WORKBOX BUILD
`yarn build`
